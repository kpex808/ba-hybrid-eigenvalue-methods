# Hybride Iterationsverfahren zur Approximation des größten Eigenwertes
ba_implement.py enthält:
- Potenzmethode
- Rayleigh-Quotient-Iteration
- Hybrid A: Zwei-Phasen-Methode
- Hybrid B: Rayleigh-basierte Gap-adaptive Methode
- Hybrid C: Rayleigh-Konvergenz-adaptive Methode
- Hybrid D: LR-optimierte adaptive Methode

ba_spektrale_lücke.py testet den Einfluss der spektralen Lücke auf Konvergenz (Test 1)
ba_robustheit.py testet Störungen im Startvektor (Test 2)
ba_kondition.py testet Konditionen der Matrix (Test 3)
ba_dimension.py testet Dimensionen der Matrix (Test 4)
